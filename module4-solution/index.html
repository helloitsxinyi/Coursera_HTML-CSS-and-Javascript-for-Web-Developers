Module 4 Solution Starter

This programme loops over an array of names and prints out either a hello or goodbye to that name to the browser console. If the name starts with a letter j or J, Goodbye JSomeName is printed. If the name starts with any other letter, Hello SomeName is printed.

The list of names given are:

  • Yaakov
  • John
  • Jen
  • Jason
  • Paul
  • Frank Larry
  • Paula
  • Laura
  • Jim
